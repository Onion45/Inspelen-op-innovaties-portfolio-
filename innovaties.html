04 — Innovaties & impact

Wat AI verandert — en wat dat betekent voor mij en PCI

AI-tools voor softwareontwikkeling zijn de afgelopen jaren in een stroomversnelling gekomen. Hieronder beschrijf ik de belangrijkste ontwikkelingen, ingedeeld naar terrein, en wat dit concreet betekent voor mijn werk en voor PCI Nederland.

Technische ontwikkelingen

AI-coding assistants (Claude Code, GitHub Copilot)

Tools die direct in je ontwikkelomgeving meekijken, code schrijven, bugs oplossen en documentatie genereren. Ik gebruik dit zelf al voor mijn eigen projecten.

Claude Code en GitHub Copilot zijn tools die direct meekijken terwijl je code schrijft. Ze vullen code aan, lossen bugs op en schrijven documentatie. Ik gebruik dit zelf al. Bij mijn project SparenVoorIedereen gebruikte ik Claude om stukken C#-code uit te leggen die ik niet direct begreep. Wat sneller ging: herhalende stukken code schrijven. Wat ik altijd controleer: of de output ook echt klopt. AI maakt fouten. Je moet het zelf blijven begrijpen.

AI in self-service portals / chatbots

Steeds meer IT-portals krijgen een chatbot of slimme zoekfunctie. Gebruikers kunnen dan zelf hun vraag stellen zonder dat er een helpdeskmedewerker aan te pas komt. Direct relevant voor mijn BPV-project bij PCI. Het portal waar ik aan werk zou hier baat bij kunnen hebben. Denk aan een chatbot die simpele vragen beantwoordt, of een systeem dat een ticket automatisch in de juiste categorie plaatst. Dat scheelt tijd voor de helpdesk.

Direct relevant voor jouw BPV-project: zou het PCI-portal hier baat bij hebben? Welke vorm (chatbot, suggesties bij tickets, automatische categorisatie)?

Commerciële ontwikkelingen

Bedrijven die AI inzetten voor klantenservice

Veel softwarebedrijven bieden inmiddels AI-functionaliteit als verkoopargument (bijv. "AI-powered support"). Dit zet ook IT-afdelingen onder druk om mee te bewegen.

Veel softwarebedrijven gebruiken AI nu als verkoopargument. "AI-powered support" staat op hun website. Dat zet ook IT-afdelingen onder druk. Klanten verwachten steeds meer dat problemen snel en automatisch worden opgelost. Voor PCI betekent dit: als concurrenten dit aanbieden en PCI niet, merken klanten dat verschil.

Concreet voor het PCI-portal project

1. Automatische ticketcategorisatie. Als een medewerker een verzoek indient, bepaalt AI automatisch om welke categorie het gaat. Dat scheelt de helpdesk tijd en zorgt dat het ticket bij de juiste persoon terechtkomt. 2. Slimme zoekfunctie. In plaats van een formulier invullen, typt de medewerker gewoon zijn probleem in. De portal zoekt dan zelf het juiste antwoord of het juiste formulier. Simpele vragen worden zo direct beantwoord zonder tussenkomst van een medewerker.
